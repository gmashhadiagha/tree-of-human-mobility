# Tree of Human Mobility

**Authors:**  
Ghazal Mashhadiagha, 
Professor Brian L. Davis  

## Description
*Tree of Human Mobility* is a data-driven artwork that visualizes mobility-related disease burden across the human lifespan. The structure of the tree was generated using a custom Python visualization pipeline that maps conditions affecting human movement into a hierarchical, radial branching system.

Each branch represents a group of conditions contributing to mobility impairment, while the overall tree metaphor reflects growth, vulnerability, and accumulation of health burden across life stages.

## Artistic Process
The computational structure of the tree was created using Python and custom visualization code developed by the authors. The resulting geometry encodes the relationships between mobility-related diseases and their impact over time.

AI-assisted tools (Leonardo AI) were then used selectively to enhance the visual texture of the trunk, roots, and natural tree elements while preserving the underlying computational structure. This approach maintains the integrity of the data-driven visualization while transforming it into a more expressive artistic form.

## Concept
The work explores the intersection of scientific visualization and symbolic representation. By framing mobility-related disease burden as a living tree, the piece emphasizes the interconnected biological systems that support human movement and how these systems change throughout the lifespan.

## Tools
- Python (custom visualization code)
- Data-driven hierarchical layout
- Leonardo AI (visual texture and artistic refinement)
